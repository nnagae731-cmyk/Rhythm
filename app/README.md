# Rhythm MVP

Expo + React Native + TypeScriptで作成した、忘れ防止・遅刻防止アプリのMVPです。

## 起動

```sh
npx expo start
```

表示されたQRコードをExpo Goで読み取ると、iPhoneまたはAndroid実機で確認できます。

## 実装済み

- タスク登録、完了、削除
- タスクと設定のAsyncStorage保存
- 到着時刻、移動・準備・余裕時間からの逆算
- 準備開始、出発10分前、最終出発時刻の通知予約
- 通知の「完了」「10分後」アクション
- ホーム画面のタスク一覧
- ウィジェット風プレビューとサイズ設定
- 完了タスク表示設定
- 寝坊防止とカスタムテーマのプレミアムロック表示

## MVP後に行うもの

- 実ホーム画面ウィジェット
- 起床確認と強い再通知
- ストア課金
- Android Studio / EAS Buildによる配布ビルド

通知はOS設定や省電力設定の影響を受けます。寝坊防止機能を追加する際も「必ず起きられる」「必ず間に合う」とは保証しない設計にします。
